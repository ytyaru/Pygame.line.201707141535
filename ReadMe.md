# このソフトウェアについて

pygameで直線の等速直線アニメーションを描いてみた。

# 開発環境

* Linux Mint 17.3 MATE 32bit
* pyenv
    * Python 3.6.1
        * pygame 1.9.3

# 実行

```sh
$ python I.py
```
